# anonymous q&a

Simple NGL-style anonymous message page.

## Run it
Just open `index.html` in a browser.

## Deploy on GitHub Pages
1. Create a new repo, upload `index.html` and `cat-avatar.png` (same folder, root of repo).
2. Go to Settings → Pages → Deploy from branch → select `main` / `root`.
3. Your site goes live at `https://yourusername.github.io/reponame/`.
